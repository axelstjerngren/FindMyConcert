FAQ

Q: How do I create an account?

Answer

Click the 'sign up' button at the top right hand side of the page. Once clicked, you will be taken to the sign up page where you will be given the option to either create a user account or a venue account. You should create a user account if you wish to search for concerts near you. A venue account should be created if you are venue which would like to host concerts on our site. Once you have created an account, you will be sent an email which contains a link that will enable you to actiavte your account. You must do this in order to access all features of FindMyConcert

Q: Can I still use FindMyConcert without an account?

Answer

Yes you can! However, the features which you can use will be limited. Without an account, you are able to view the full list of concerts and also search for them too. You can search by your current location using the blue button to the right of the search bar. This will search for concerts near you! Plus, you will be able to view the concert by hovering over the tile and clicking the 'view' button.

Q: How do I see more information about a concert?

Answer

Hover over one of the tiles on the events page and watch it flip around! On the back, you will be greeted by a button that says 'view'. Click this and you will be taken to a page which is filled with information about the event. On this page, you will have the ability to:

  • View and post comments
  • Rate the event and see its average rating
  • See more details about the venue
  • Use the Spotify widget in order to play songs about the artist
  • See the location of the venue on a google map

Q: How do I add an event of my own?

Answer

Provided that you have created and signed into a venue account, you can add a concert very simply. Click the 'add concert' button at the top of the page once you have logged in. You will be greeted with a simplistic form which enables you to add a concert easily and efficiently. Once added, your concert will appear as a tile on the events page.

Q: I've forgotten my password, how do I reset it?

Answer

Make sure you are signed out and click the 'sign in' dropdown menu in the top right hand corner of the page. Once here, clicked the 'Forgot password?' button at the bottom. Enter the email address associated with your account and follow the instructions sent to your email address in order to reset your password. Please note, if you enter an email address which doesn't relate to an account on our system then no password reset email will be sent to you!

Q: How does discover work?

Answer

The discover page recommends you artists which you have not yet bookmarked. It recommends these artists by correlating your ratings with other users who have rated concerts in a similar fashion as you. The recommender engine uses a mathematical operation called singular value matrix decomposition (henceforth SVD). All user ratings are stored in a matrix where each row represents a unique user and each column represents a unique artist. Missing ratings (all occurrences where a user has not rated a user) are called latent factors. Most ratings in the matrix are latent factors, so the matrix is a sparse matrix.

In SVD the ratings matrix (xÌ‚) is decomposed into three lower dimensional matrices. U is the matrix which represents the relationship between users and latent factors. S is a diagonal matrix which (essentially) represents the weights of each rating. The last matrix VT is a right singular matrix, this represents the relationship between artists and the latent factors. We decompose the ratings matrix with a rank K = 5 and then multiply U, S and VT back together to get the rank K approximation of the ratings matrix. Now all the latent factors are filled in with a value whcih should approximately represent the rating each user would have given the artist in question. Now a row which represents a user can be extracted, transposed and sorted to find teh aritists which the user might enjoy.
